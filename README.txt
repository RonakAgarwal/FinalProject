App Description
Paper Trading app that allows users to have unlimited money to virtually test how their stock portfolios would do if they actually invested them. Stores holdings in a database and uses real time stock data to evaluate them. Also has a page to check the latest stock prices and their price trends. 
Link: https://three35-stock-app.onrender.com/
